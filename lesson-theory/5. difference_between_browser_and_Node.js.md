# Різниця між браузером і Node.js

Node.js і браузер обидва виконують JavaScript, але вони створені для різних цілей, 
мають різні середовища та можливості.

Порівняльна таблиця (відкривається окремо в папці, або в режимі перегляду .md файлу):

![The difference between a browser and Node.js](/images/difference_between_browser_and_Node.js.png)

### 1. Глобальний об’єкт

- У **браузері**: Глобальний об’єкт — це window.
```js
console.log(window); // Глобальний об'єкт браузера
```
- У **Node.js**: Глобальний об’єкт — це global.
```js
console.log(global); // Глобальний об'єкт Node.js
```
Приклад:
```js
// Браузер
console.log(window.alert); // Є функція alert
console.log(global); // Undefined

// Node.js
console.log(global.setTimeout); // Є функція setTimeout
console.log(window); // ReferenceError: window is not defined
```

### 2. Доступ до DOM

- У **браузері** є доступ до DOM для маніпуляції HTML і CSS.
```js
document.querySelector('h1').textContent = 'Привіт, Світ!';
```
- У **Node.js** DOM не існує, тому подібні операції неможливі.
```js
// Браузер
console.log(document.body); // Доступ до елементів DOM

// Node.js
console.log(document); // ReferenceError: document is not defined
```

### 3. Доступ до файлової системи

- У **браузері** ви не можете безпосередньо читати чи писати файли на диску користувача через обмеження безпеки.
- У **Node.js** є модуль fs для роботи з файлами.
```js
const fs = require('fs');
const data = fs.readFileSync('example.txt', 'utf8');
console.log(data);
```
Приклад:
```js
// Node.js: Читання файлу
const fs = require('fs');
fs.readFile('example.txt', 'utf8', (err, data) => {
  if (err) throw err;
  console.log(data);
});
```

### 4. Мережеві запити

- У **браузері** HTTP-запити виконуються через fetch або XMLHttpRequest.
```js
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data));
```

- У **Node.js** використовується модуль http або бібліотеки (axios, node-fetch).
```js
const https = require('https');

https.get('https://api.example.com/data', res => {
  res.on('data', chunk => {
    console.log(chunk.toString());
  });
});
```

### 5. Асинхронність

І **браузер**, і **Node.js** підтримують асинхронні операції через Promises та async/await.
```js
// Асинхронний таймер
setTimeout(() => {
  console.log('Пройшло 2 секунди');
}, 2000);
```

### 6. Модулі

- У **браузері** використовується стандарт ES-модулів (import/export).
```js
import { greet } from './greet.js';
greet('Світ');
```

- У **Node.js** підтримуються як CommonJS (require/module.exports), так і ES-модулі:
```js
const greet = require('./greet');
greet('Світ');
```

### Висновок

Браузер:

	•	Призначений для роботи з користувачем (HTML, CSS, DOM).
	•	Має доступ до інструментів UI, таких як window, document.

Node.js:

	•	Призначений для серверних задач (робота з базами даних, файлами, мережами).
	•	Має доступ до системних ресурсів (файлова система, мережа).
