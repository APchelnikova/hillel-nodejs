### Приклад 2: Читання параметрів із командного рядка
Node.js дозволяє передавати параметри через термінал.

Створіть файл **greet.js**:
```js
const name = process.argv[2]; // Отримуємо перший аргумент
console.log(`Привіт, ${name || 'Користувач'}!`);
```
process.argv:
- process — це глобальний об’єкт у Node.js, який надає інформацію про поточний процес.
- process.argv — це масив, що містить аргументи, передані при запуску програми.
- Структура process.argv:
- process.argv[0] — шлях до Node.js.
- process.argv[1] — шлях до виконуваного файлу.
- process.argv[2] і далі — параметри, передані в командному рядку.

Наприклад, якщо ми виконаємо команду:
_node greet.js Катя_

То масив process.argv виглядатиме так:
```js
[
  '/path/to/node',        // Node.js
  '/path/to/greet.js',    // Скрипт greet.js
  'Вова'                  // Параметр, переданий у командному рядку
]
```

Запустіть у терміналі:
_node greet.js Катя_

Отримаєте:
_Привіт, Катя!_

Якщо не передати параметр, то виведе:
_Привіт, Користувач!_


### Приклад 3: Власна функція

Створіть файл **3. functions.js**:
```js
function greet(name) {
  return `Привіт, ${name}!`;
}

console.log(greet('Світ'));
console.log(greet('Node.js'));
```

Запустіть файл:
_node functions.js_

Результат:
_Привіт, Світ!
Привіт, Node.js!_


### Приклад 4: Виведення випадкового числа

Створіть файл **4. random.js**:
```js
const min = 1;
const max = 100;
const randomNumber = Math.floor(Math.random() * (max - min + 1)) + min;

console.log(`Випадкове число між ${min} і ${max}:`, randomNumber);
```

Запустіть файл:
_node random.js_

Результат (наприклад):
_Випадкове число між 1 і 100: 37_


### Приклад 5: Робота з текстом

Створіть файл **text.js**:
```js
const text = 'Node.js — це чудова платформа для серверного програмування!';
console.log('Довжина тексту:', text.length);
console.log('Верхній регістр:', text.toUpperCase());
console.log('Чи містить "Node.js"?', text.includes('Node.js'));
```

Запустіть файл:
_node text.js_

Результат:
_Довжина тексту: 55
Верхній регістр: NODE.JS — ЦЕ ЧУДОВА ПЛАТФОРМА ДЛЯ СЕРВЕРНОГО ПРОГРАМУВАННЯ!
Чи містить "Node.js"? true_


### Приклад 6: Простий цикл

Створіть файл **loop.js**:
