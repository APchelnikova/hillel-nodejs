# Встановлення Node.js

На цьому етапі навчання у вас у всіх має бути встановлений Node.js на ваші компʼютери, але якщо ні -

### Кроки для встановлення:

1. Перейдіть на офіційний сайт Node.js. 

https://nodejs.org/en

2.	Завантажте LTS (Long Term Support) версію.
3.	Виконайте інсталяцію, натискаючи “Next”.

### Перевірка встановлення:
Після встановлення відкрийте термінал і виконайте:

node -v   # Перевіряє версію Node.js
npm -v    # Перевіряє версію npm


Успішний результат виглядатиме приблизно так:
v18.12.1
9.6.2

Якщо все як тут, то вітаю, ви встановили Node.js.
