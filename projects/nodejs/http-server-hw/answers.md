## 1. Чому при відкритті `http://localhost:3000` не потрібен DNS?

- `localhost` — це спеціальне зарезервоване ім'я, яке операційна система знає заздалегідь.
- Його IP-адреса — **127.0.0.1**, завжди вказує на локальний комп'ютер.
- Тому запит надсилається на локальну машину без використання DNS.

---

## 2. Що станеться, якщо запустити два сервери на одному порту 3000?

Другий сервер не запуститься, оскільки порт **3000** вже зайнятий першим сервером.

**Текст помилки:**

```text
Error: listen EADDRINUSE: address already in use :::3000
```

---

## 3. Для чого потрібен заголовок `Content-Type`? Що зміниться, якщо його прибрати?

- Заголовок `Content-Type` повідомляє клієнту, який тип даних передається (наприклад, HTML, JSON або текст).
- Якщо його прибрати, клієнт може неправильно визначити тип вмісту та некоректно його обробити або відобразити.

---

## 4. У якому місці коду визначається, який код стану поверне сервер?

Код стану визначається під час формування відповіді сервером, зокрема у виклику:

```javascript
response.writeHead(statusCode, headers);
```

Наприклад:

```javascript
response.writeHead(200, {
  "Content-Type": "text/html",
});
```

Тут сервер повертає код стану **200 OK**.
